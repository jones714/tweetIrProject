Twitter sentiment analysis

Result Analysis

 Problems We Faced

We weren't able to receive a Twitter API Key in a timely matter which resulted in the usage of the Twpy library. Twpy allowed us to gather tweets based on a query search that we used for our project. If we were able to use the Twitter API Key instead we would have access to better data.

Accuracy

As mentioned, the data collected from twitter was collected using Twpy. When we call twpy it only grabs a subset of tweets from an overall pool of tweets, so each time you run the anaylsis it will result with different data. Given this it is not possible to calculate Precision/Recall and Accuracy if we can not count our True Positives/True Negatives etc. Through the mind of a person one can tell the sentiment analysis has yet to understand sarcasm as it usually counts it as neutral or mistakeningly positive or negative.

Future Optimizations

Once receiving an API Key the program will have stronger data to come up with a stronger analysis. In the future the project could implement filters to make sure the tweets are better quality and only look for tweets that aren't just pictures. Machine learning is also a good future optimization as it will allow the analysis to better understand human sarcasm.
